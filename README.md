# Chemistry 202 Lab Practice

Interactive practice website for the Chemistry 202 lab summary.

## What is included

- Six experiment sections.
- 184 mixed practice questions.
- Shuffled answer choices.
- Correct/wrong highlighting with explanations.
- Review mode for missed questions.
- Flashcards for quick concept review.
- Missed-concepts summary.
- Formula practice mode with numeric checking.

## How to use locally

Open `index.html` in a browser.

No build step or server is required.

## Files

- `index.html` - page structure.
- `styles.css` - visual design.
- `data.js` - question bank, flashcards, and formula drills.
- `app.js` - quiz logic and progress state.
